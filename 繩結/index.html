📌 上攀步驟
1
確保（胸繩 + 定位）：先上、最後解。1.2m 普魯士打好後，扣上胸繩往上吐一點再往下坐（定位防撞岩壁）
2
往下坐成懸空狀態後，再綁 2m 腳繩（懸空狀態下打普魯士結）
3
上攀：腳往下踩 + 左手伸直往上抓主繩，使身體垂直，省力。胸繩結上推不超過頭頂
4
下降：腳繩先往上踩一點；胸繩下推不超過肚臍上一拳；腳繩下推不超過膝蓋（大腿左右）
⚠️ 普魯士移動方式：抓靠近繩結的雙股繩，提起同高後往主繩方向「吐」 → 繩結鬆開可移動
吐完後往下拉確認咬緊再移動。不要抓繩結本身。

上升轉下降（受力轉換）
三種方法：傳統 / 克式+半扣 / 過繩結
📌 方法1：傳統上升轉下降
1
1.2m 胸繩（普魯士）→ 懸空 → 上 2m 腳繩（普魯士）→ 上攀
2
放雙手 + 腳拿出繩圈後，在腳繩下方上下降器（ATC or 8字環）
⚠️ 若同一確保環，下降器要在胸繩環的外面
3
制動繩由內而外纏右腳 5圈（纏緊到腿痛），繩尾穿過 2–3 圈後打單滑結 + 單結鎖緊
4
踩腳繩往上，將胸繩往下拉使之不受力 → 解腳繩 → 解胸繩（可解可不解）
5
（ATC）左手抓緊 ATC 壓繩，右手解大腿五圈下降
（8字環）左手抓 8字下面制動繩，右手解大腿五圈下降
📌 方法2：克式 + 半扣上升轉下降（較快）
1
2m 胸繩（克式+半扣）→ 懸空 → 上 2m 腳繩（普魯士）→ 上攀
2
放雙手後解腳繩 → 另取 D鉤 + 下降器扣上吊帶確保環（同一個：扣在外面）
3
右手抓緊主繩靠近髖部制動 + 左手兩指輕推克式 + 掌根輕抵半扣直到全鬆弛，即可下降
📌 方法3：過繩結🪢 上升轉下降（最複雜）
上升遇到繩結：
1
在繩結🪢上方打克式+半扣+掛 D鉤
2
踩腳繩往上撐，將克式+半扣的 D鉤掛入確保環（同一個：下降器在外面）
3
解腳繩，在繩結🪢上方打普魯士結 → 繼續上攀
下降遇到繩結：
1
在胸繩下方上 ATC（同一確保環：ATC 在外面）
2
確認 1.7m 克式+半扣受力不太緊 → 解腳繩 → 克式+半扣配合 ATC 下降到繩結🪢
3
解 ATC，換到繩結🪢下方 → ATC 上打 Hot lock（⚠️ 打在繩結下方、ATC 上方）
4
解胸繩（換 ATC+Hot lock 受力）→ 解 Hot lock → ATC 制動下降
⚠️ 時時確認 D鉤無橫置；克式+半扣繩結型態有無亂；Hot lock 不要打在繩結🪢上（受力後很難解開）
Hot Lock（ATC 動態制動）：左手壓住 ATC 制動繩 + 右手拿制動繩成繩圈穿進 ATC 的 D鉤，「往外轉半圈壓繩」後從繩圈內拿出繩尾 → 拉緊靠緊 ATC → 在受力繩端打單結鎖緊（= 在 D鉤內打 MMO）
受力後卡住解法：① 固定點預先打義大利半扣 + MMO → 可解固定點制動放人下來；② LRH（負重釋放系統）

固定點建立
SERENE-A × 角度 × 均力系統
SERENE-A：Solid（穩固）· Equalized（均力）· Redundant（備份）· Efficient · No Extension · Angles < 60°
⚠️ 固定點角度：兩支點夾角 ≤ 60°（最好），絕不超過 90°
手勢量角：🇯🇵 日本7 = 90°、Rocker = 60°、✌️ = 30°、六 = 120°
120° 時每個支點承受 100% 的力（= 完全沒分擔）
「雞蛋不要放同一個籃」：不同固定點、越多越好
可以繞固定點就不要打栓牛（栓牛讓繩受力減少 50–70%）
限位結：減少墜落長度（人體無法承受超過 50cm 的墜落衝擊），使不同固定點間受力均勻
樹幹固定點
細樹叢/鋼筋：雙股雙套，留一個繩圈打確保繩
粗樹幹：sling 打栓牛結 + D鉤，確保打在 D鉤上
雙股雙套結 + 單結收尾 or sling 打栓牛結 + D鉤 + 主繩在 D鉤上打雙8
均力支點系統
兩固定點各繞輔助繩 + D鉤
→ 1.2m or 1.8m sling 兩端掛 D鉤（避開接縫）
→ 取其中一股順/逆轉半圈
→ 在兩股最低處掛另一個 D鉤
三個固定點：取其中兩股各轉半圈
sling 長度建議
sling 最好 < 1.2m（長了可縮短）
避開接縫處掛 D鉤

雙套結Clove Hitch
固定點快速
將繩子繞過支點（柱、D鉤）兩圈，第二圈壓在第一圈上面，形成交叉 → 拉緊兩端

打在 D鉤上（義大利半扣方式）：
① 右手正手翻半圈形成第一個繩圈
② 同向再翻半圈形成第二個繩圈
③ 第二圈疊在第一圈前方 → 兩圈一起扣入 D鉤
  • 可在受力後微調長短
  • 建立固定點時快速連結支點
  • 單獨使用須加安全結（單結）收尾
  • 細柱上容易滑動，需配合其他結使用
≈ 雙套 vs 栓牛：正向手 = 雙套；反向手 = 栓牛
雙稱人結Double Bowline
安全結吊帶
比單稱人結多加一個保護圈，使繩結不易鬆脫，不需要額外打安全結

步驟：
① 繩端打一個小繩圈（兔子洞）
② 繩尾穿出小繩圈（兔子出洞）
③ 再穿回同一個繩圈一次（兔子又回洞）= 形成雙重保護
④ 繩尾繞過背後繩（大樹）後再穿入繩圈收緊
  • 比單稱人結安全，不需安全結
  • 固定點、吊帶連結皆適用
  • 受力後較易解開（比8字結易解）
  • 初學較難判斷結構是否正確

義大利半扣詳解
正手/反手 × 打在自己身上 vs 固定點
核心原則：繩索放入鉤環時，以「受力繩」判斷正反向
· 受力繩在外側（遠離岩壁） → 正手為正向
· 受力繩在裡側（靠近岩壁） → 反手為正向

⚠️ 攀岩時絕對不可讓非受力繩（制動繩）在 D鉤開口那側！（Fall 時可能衝出）
打在自己身上
扣 D鉤：右手正拿大D鉤，由右往左橫向扣進吊帶直向繩圈，再逆時針繞半圈使大D寬頭朝外

右手正手打：「用D鉤去扣繩」→ 右手正手翻（由內往外、由下往上）半圈形成繩圈 → 繩圈繞到左邊D鉤開口處卡進去

左手反手打：「用D鉤去扣繩」→ 左手反手翻（由外往內、由下往上）半圈形成繩圈 → 直接往下扣進左邊D鉤開口處
打在固定點上
8字結打自己身上，另一頭綁固定點

受力繩在前面（D鉤開口朝右，大頭朝下）：
右手拿受力繩正手翻半圈 → 繩圈由右側掛入D鉤開口（繩包繩：受力繩被包在中間）

受力繩在後面：
右手拿受力繩反手翻半圈 → 繩圈由右側掛入D鉤開口
⚠️ 打半扣時受力繩都不要繞過非受力繩（制動繩）
